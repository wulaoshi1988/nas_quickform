# QuickForm - 智能表单分析与报告生成系统

## 项目定义
QuickForm是一款基于Web的智能表单管理服务，旨在简化数据收集、分析和报告生成的全流程。它允许用户创建自定义表单，收集结构化数据，并利用AI技术自动分析数据模式、生成可视化图表和提供深度洞察，最终输出专业的HTML格式分析报告。

## 项目简介
QuickForm是一个由温州科技高级中学和温州大学联合开发的表单服务信息系统，实现数据收集、分析与报告生成，专注于提供智能的数据分析和可视化功能。系统支持创建自定义表单、收集用户提交的回答、通过AI进行深度分析，并自动生成详细的分析报告。

## 主要功能

### 1. 表单管理
- 创建自定义表单，支持多种问题类型
- 编辑和管理现有表单
- 查看表单详情和统计信息

### 2. 数据分析
- 智能分析用户提交的表单数据
- 自动识别趋势、模式和关键洞察
- 生成可视化图表和数据摘要

### 3. 报告生成
- 根据分析结果自动生成详细的HTML格式报告
- 支持自定义报告模板和样式
- 报告支持下载和保存

### 4. 智能分析
- 利用AI技术进行深度数据分析
- 提供智能洞察和建议
- 支持多种分析维度和方法

## 技术栈

- **后端**: Python, Flask
- **数据库**: SQLite
- **前端**: HTML, CSS, JavaScript, Bootstrap
- **AI集成**: 支持多种AI模型接口
- **部署**: 本地开发服务器或生产环境部署

## 快速开始

### 环境要求
- Python 3.7+
- pip包管理器

### 安装步骤

1. **克隆项目**
   ```bash
   git clone [项目仓库地址]
   cd QuickForm
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **配置环境变量**
   - 复制`.env.example`文件为`.env`
   - 修改`.env`文件中的配置项

4. **初始化数据库**
   ```bash
   python app.py
   ```

5. **启动服务器**
   ```bash
   python app.py
   ```

6. **访问系统**
   - 打开浏览器访问: http://localhost:5001/

## 使用说明

### 创建和管理表单
1. 登录系统后，在控制面板点击"创建新表单"
2. 设置表单标题、描述和各种问题
3. 保存表单并获取表单链接

### 查看和分析数据
1. 在表单列表中选择需要分析的表单
2. 点击"分析数据"按钮
3. 系统将自动处理数据并生成分析结果

### 生成报告
1. 在分析页面，点击"生成报告"
2. 系统将自动创建详细的分析报告
3. 可以查看、下载或重新生成报告

## 配置说明
